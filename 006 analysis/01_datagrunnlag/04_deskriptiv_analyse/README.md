# 04 Deskriptiv analyse

Denne aktiviteten inneholder skript og artefakter for deskriptiv analyse av datagrunnlaget.

- `src`: skript for å lage figurer
- `figures`: genererte figurer til rapport og analyse
- `metadata`: figurguide og forklaringer

Formålet er å støtte casebeskrivelse og datakapittel med konsistente visualiseringer.
