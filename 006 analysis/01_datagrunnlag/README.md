# 01 Datagrunnlag

Denne hovedmappen følger aktivitetsrekken i prosjektplanen for datagrunnlaget:

- `01_datainnsamling`
- `02_datavask`
- `03_strukturering_av_datasett`
- `04_deskriptiv_analyse`

Formålet er å gjøre det tydelig hvor rådata brukes, hvor rensing skjer, hvor strukturert datasett etableres, og hvor deskriptive figurer og tabeller produseres.
