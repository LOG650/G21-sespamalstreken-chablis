# 01 Datainnsamling

Denne aktiviteten dokumenterer hvilke rådata som er brukt som grunnlag for videre analyse.

Per nå ligger den faktiske råfilen fortsatt i `004 data`, mens denne mappen fungerer som planmessig plassholder for notater og eventuell dokumentasjon av datainnsamlingen.

Tilleggsdata mottatt etter første datavask er dokumentert i `tilleggsdata_2025.md`. Dette gjelder åtte anonymiserte fartøyfiler fra 2025 med forbruk, voyage-koder, havner, ROB-felt, oppgitt bunkerskapasitet per fartøyklasse og kontraktsinformasjon for Singapore, Sør-Korea og Rotterdam.
