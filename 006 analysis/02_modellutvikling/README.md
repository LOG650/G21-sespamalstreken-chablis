# 02 Modellutvikling

Denne hovedmappen følger aktivitetsrekken i prosjektplanen for modellutvikling:

- `01_definere_variabler`
- `02_formulere_malfunksjon`
- `03_definere_restriksjoner`
- `04_implementere_modell`
- `05_teste_modell`

Prosjektets operative hovedmodell ligger i `04_implementere_modell`. Aktivitetene før dette dokumenterer variabler, målfunksjon og restriksjoner for samme modell, slik at modellutviklingen peker mot én felles implementasjon.
