# 04 Implementere modell

Denne aktiviteten inneholder den første kjørbare modellimplementasjonen.

- `src`: skript for å generere modellinput og kjøre Pyomo-modellen
- `input`: modellinput og parameterfiler
- `output`: løsningsfiler fra faktisk modellkjøring

Modellversjon 1 beholdes i filnavnene, men mappestrukturen følger aktivitetene i prosjektplanen.
